# 完整表单实施指南

根据三份配置表（工艺类、设备类、材料类），本文档提供完整的实施方案。

## 一、整体架构

### 1.1 填报流程
```
步骤1：基础信息（姓名、手机、邮箱、单位、信用代码、法人、成立时间）
  ↓
步骤2：选择体现形式（工艺类/材料类/设备类）+ 技术名称
  ↓
步骤3：详细信息（根据选择的类型动态显示）
  - 工艺类：专业领域 → 动态字段
  - 材料类：专业应用领域 → 功能分类 → 应用行业 → 成熟度 → 获奖情况
  - 设备类：设备类别 → 专业领域 → 动态字段
  ↓
步骤4：确认提交
```

### 1.2 级联逻辑总结

#### 工艺类
```
专业领域（6选1）
  ↓
工艺颗粒度（4选1，必填）
  ├→ 选择后三项 → 显示"工艺环节"
  ↓
工艺原理（多选，必填）
  ├→ 选择"其他" → 显示填写框
  ↓
成熟度（5选1，选填）
  ├→ 选择后三项 → 显示"案例简介"
  ↓
技术来源（3选1，选填）
  ↓
应用行业（可添加多个，必填）
  ↓
工艺特点（多选，必填，仅水处理有）
  ├→ 11个带填写框的选项
  ├→ 11个简单勾选
  ├→ "其他" → 填写框
  ↓
污染物治理对象（多选，必填，根据领域不同）
  ├→ 水处理：常规/重金属/特殊/新污染物/非常规/其他
  ├→ 大气：常规/非常规/其他
  ├→ 固废：类型/其他
  ├→ 生态：污染治理型/生态恢复型/资源保护型/灾害防治型/景观美化型
  ├→ 土壤：无机/重金属/挥发性/半挥发性/新污染物/其他
  ├→ 节能：类型/其他
  ↓
获奖情况（是/否，选填）
  ├→ 选择"是" → 显示奖项级别（多选）+ 获奖等级（单选）
```

#### 材料类
```
专业应用领域（6选1）
  ↓
功能分类或工艺环节（多选，必填，根据领域显示）
  ├→ 水处理：13个选项
  ├→ 大气：除尘/脱硫/脱硝/脱VOCs（带二级）/滤料/催化/吸收/吸附/其他
  ├→ 固废：17个选项
  ├→ 生态工程：20个选项
  ├→ 土壤修复：6个选项
  ├→ 节能与碳减排：同大气
  ↓
应用行业（可添加多个，必填）
  ↓
成熟度（5选1，选填）
  ├→ 选择后三项 → 显示"案例简介"
  ↓
获奖情况（是/否，选填）
  ├→ 选择"是" → 显示奖项级别 + 获奖等级
```

#### 设备类
```
设备类别（治理设备/检测设备，必填）
  ↓
  ├→ 治理设备：
  │     ↓
  │   专业领域（6选1）
  │     ↓
  │   污染物类别（多选，必填，根据领域显示）
  │     ↓
  │   设备特点/适用环境（多选，选填，根据领域显示）
  │     ↓
  │   应用行业（可添加多个，必填）
  │     ↓
  │   来源（国内/海外，多选，选填）
  │     ↓
  │   成熟度（5选1，选填）
  │     ├→ 选择后三项 → 显示"案例简介"
  │     ↓
  │   获奖情况
  │
  └→ 检测设备：
        ↓
      专业领域（4选1：水处理/大气/水生态修复/其他）
        ↓
      检测内容（根据领域显示）
        ↓
      应用行业（可添加多个，必填）
        ↓
      来源（国内/海外，单选，选填）
        ↓
      成熟度（5选1，选填）
        ├→ 选择后三项 → 显示"案例简介"
        ↓
      获奖情况
```
