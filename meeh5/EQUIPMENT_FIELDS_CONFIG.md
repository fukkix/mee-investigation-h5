## 填写说明

### 级联逻辑总结

1. **第一级**：设备类别（治理设备 / 检测设备）
2. **第二级**：专业领域（水处理 / 大气 / 固废等）
3. **第三级**：根据专业领域显示对应的：
   - 污染物类别（治理设备）或检测内容（检测设备）
   - 设备特点/适用环境
4. **第四级**：应用行业、来源、成熟度
5. **第五级**：案例简介（成熟度后三项时显示）
6. **第六级**：获奖情况

### 注意事项

1. 设备类的级联比工艺类多一层（设备类别）
2. 治理设备和检测设备的字段结构不同
3. 检测设备的专业领域选项比治理设备少（没有固废、土壤修复、节能与碳减排）
4. 大气污染治理的检测设备使用文本框而非多选框
