## 填写说明

### 1. 字段类型标注
- **单选**：用户只能选择一个选项
- **多选**：用户可以选择多个选项
- **必填**：该字段必须填写
- **选填**：该字段可以不填

### 2. 级联规则说明
请明确说明：
- **显示条件**：什么情况下显示该字段
- **级联规则**：选择某个选项后会触发什么变化

### 3. 二级菜单说明
对于带二级菜单的选项，请用 `→` 标注，并说明：
- 填写框类型（文本框/范围输入/数字输入）
- 占位符文本
- 验证规则（如果有）

### 4. 父子选项说明
对于有父子关系的选项（如污染物分类），请用缩进表示层级关系。

### 5. 共用字段
如果某些字段在所有领域都相同（如工艺原理、技术来源、应用行业、成熟度、获奖情况），可以标注"同水处理"，我会自动复用。

## 完成后的处理流程

1. 您填写完整的配置文档
2. 我根据配置生成 JavaScript 配置对象
3. 我实现通用的动态表单生成函数
4. 我实现级联逻辑处理函数
5. 我更新表单验证和数据提交逻辑

这样可以确保所有领域的逻辑一致，且易于维护和扩展。
