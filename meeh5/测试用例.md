## 通用说明

| 符号 | 含义 |
|------|------|
| ✅ | 预期通过 |
| ❌ | 预期失败/报错 |
| 📝 | 需要记录结果 |

## 一、通用功能

### TC-001 步骤导航与滚动定位

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| 1 | 打开页面 | 显示步骤1，顶部显示"步骤 1/4" |
| 2 | 不填任何内容，点击"下一步" | 提示"请填写所有必填项"，必填框变红边框，不跳转 |
| 3 | 填写步骤1所有必填项，点击"下一步" | 跳转步骤2，页面**平滑滚动**到"体现形式"选择区 |
| 4 | 在步骤2点击"上一步" | 返回步骤1，页面滚动到"姓名"输入框 |
| 5 | 步骤2填完，点击"下一步" | 跳转步骤3，页面滚动到"专业领域"下拉框 |
| 6 | 步骤3填完，点击"下一步" | 跳转步骤4，页面滚动到确认信息区域 |

## 二、工艺类

### TC-101 水处理 - 完整流程

**前置**：步骤1填完 → 步骤2选"工艺类"，填写工艺名称

#### 2.1 专业领域选择
| 操作 | 预期结果 |
|------|---------|
| 下拉选择"水处理" | 动态显示：工艺颗粒度、工艺原理、成熟度、技术来源、应用行业、工艺特点、污染物治理对象、获奖情况 |

#### 2.2 工艺颗粒度 → 工艺环节级联
| 操作 | 预期结果 |
|------|---------|
| 选择"综合型工艺系统流程" | 工艺环节区块**不显示** |
| 改选"特定功能型工艺流程" | 工艺环节区块**显示** |
| 改选"处理单元型工艺" | 工艺环节区块**显示** |
| 改选"其他" | 工艺环节区块**显示** |

#### 2.3 工艺原理
| 操作 | 预期结果 |
|------|---------|
| 勾选"物理法"、"生物法" | 正常多选 |
| 勾选"其他" | 显示填写框 |
| 取消勾选"其他" | 填写框隐藏，内容清空 |

#### 2.4 成熟度 → 案例简介级联
| 操作 | 预期结果 |
|------|---------|
| 选择"已开展中试，运行稳定" | 案例简介**不显示** |
| 选择"完成中试研究，拟开展示范工程" | 案例简介**不显示** |
| 选择"完成示范工程，运行情况待观察" | 案例简介**显示**，字段变必填 |
| 选择"建成2~4项工程且运行稳定" | 案例简介**显示** |
| 选择"建成5项及以上工程项目的" | 案例简介**显示** |
| 案例简介显示后，改选前两项 | 案例简介**隐藏**，必填取消 |

#### 2.5 应用行业
| 操作 | 预期结果 |
|------|---------|
| 填写行业代码、大类、中类、小类 | 正常输入 |
| 点击"添加行业" | 新增一行输入框，带删除按钮 |
| 点击删除按钮 | 该行移除 |

#### 2.6 工艺特点（水处理专有）
| 操作 | 预期结果 |
|------|---------|
| 勾选"水质波动幅度宽" | 显示"污染物类型及幅度范围"填写框 |
| 勾选"温度波动幅度宽" | 显示"范围：___至___"填写框 |
| 取消勾选 | 填写框隐藏，内容清空 |
| 勾选"占地集约化"等简单选项 | 直接勾选，无填写框 |
| 勾选"其他" | 显示填写框 |

#### 2.7 污染物治理对象（折叠分组）
| 操作 | 预期结果 |
|------|---------|
| 初始状态 | 只显示"常规污染物"、"重金属污染物"、"特殊污染物"三个折叠按钮，以及新污染物、非常规污染物、其他 |
| 点击"常规污染物" | 展开16个子选项，箭头旋转180° |
| 再次点击"常规污染物" | 收起子选项，箭头复原 |
| 展开后勾选子选项 | 正常多选 |
| 点击"特殊污染物"展开后勾选"其他_____等" | 显示填写框 |
| 勾选"新污染物" | 显示填写框 |
| 勾选"非常规污染物" | 显示填写框 |

#### 2.8 获奖情况 → 获奖详情级联
| 操作 | 预期结果 |
|------|---------|
| 默认选"否" | 获奖详情**不显示** |
| 改选"是" | 显示奖项级别（多选）和获奖等级（单选） |
| 勾选"国家级"、"省部级" | 正常多选 |
| 选择获奖等级"一等奖" | 正常单选 |
| 改回"否" | 获奖详情隐藏，已选内容清空 |
